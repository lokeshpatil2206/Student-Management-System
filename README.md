# 🎓 Student Management System

A menu-driven Student Management System developed in **C++** using **Object-Oriented Programming (OOP)** concepts.

## ✨ Features

- Add Student
- Display All Students
- Search Student
- Update Student Details
- Delete Student
- Menu-driven Interface
- File Handling (if implemented)

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming
- Visual Studio Code

## 📂 Project Structure

```
Student-Management-System/
│── main.cpp
│── version1.cpp
│── .gitignore
│── README.md
```

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/lokeshpatil2206/Student-Management-System.git
```

2. Compile

```bash
g++ main.cpp -o student
```

3. Run

```bash
./student
```
